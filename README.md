# EmployeeListing
This project showcases the use of simple public apis for tasks like: 

- Logging in a user
- Fetching a list of users
- Showing detail screens of a specific user

The project is in MVVM.
It's written in Swift 5
It displays all the users in a scrollable page with using tableview.
It shows detail of the user in a separate screen upon individual user item selection.
Includes pagination.
Includes Unit testing
